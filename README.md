## 🎯 快速开始

### 在线访问
直接访问 GitHub Pages：
```
https://bugcodes.github.io/ai-tools-hub/
```

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/bugcodes/ai-tools-hub.git

# 进入目录
cd ai-tools-hub

# 用浏览器打开 index.html 即可
open index.html
```

### 添加新工具
编辑 `data/tools.json`，按格式添加工具信息：
```json
{
  "name": "工具名称",
  "url": "https://官网链接",
  "description": "简短描述",
  "pricing": "免费/$20/月",
  "tags": ["标签 1", "标签 2"],
  "rating": 5,
  "affiliateUrl": "联盟链接（可选）"
}
```
